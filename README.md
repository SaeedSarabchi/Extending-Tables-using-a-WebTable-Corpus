# Extending Tables Using a Web Table Corpus

## Summary
Python research project from my M.Sc. thesis at the University of Alberta. The system extends input tables using a web table corpus and improved baseline performance by 2.5x.

## Problem
Given a seed table, identify related columns from a large web table corpus.

## Approach
- Table matching
- Candidate column retrieval
- Ranking
- Evaluation against baseline methods

## Results
- Improved state-of-the-art baseline by 2.5x based on extended columns per table

## Tech Stack
Python, data processing, information retrieval
